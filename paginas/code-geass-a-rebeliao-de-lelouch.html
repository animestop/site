Code Geass: A Rebelião de Lelouch

por Bie — 22 de agosto de 2025
Anime • Mecha • Militar • Drama • Ação • Sci-Fi

Code Geass é um thriller psicológico e militar que se passa em uma realidade alternativa onde o mundo é dominado pelo Império da Britannia. A trama segue Lelouch vi Britannia, um príncipe exilado que vive disfarçado no Japão. Após um encontro com a misteriosa C.C., ele adquire o Geass, um poder que lhe permite dar uma ordem absoluta a qualquer pessoa.

Com essa nova habilidade, Lelouch assume a identidade do mascarado Zero e cria os Cavaleiros Negros, uma força rebelde dedicada a destruir Britannia e criar um mundo melhor para sua irmã, Nunnally. A série é um jogo de xadrez em escala global, cheio de reviravoltas, dilemas morais e batalhas de mechas (Knightmare Frames) de tirar o fôlego.

Code Geass questiona constantemente o espectador: os fins realmente justificam os meios? Com um dos finais mais icônicos e debatidos da história dos animes, é uma obra essencial para qualquer fã.
